# PairFold paper — Overleaf upload

## Upload
1. Go to [overleaf.com](https://www.overleaf.com) → **New Project** → **Upload Project**
2. Select `PairFold_overleaf.zip` (one level up from this folder), **or**
3. Create a blank project and upload every file in this folder

## Compile
- Set the main document to `main.tex`
- Compiler: **pdfLaTeX** (default)
- Click **Recompile**

## Contents
| File | Role |
|------|------|
| `main.tex` | Full paper (same structure as before + expanded benchmarks) |
| `macros_domains.tex` / `macros_long.tex` | Measured Domains100 / Long100 numbers |
| `table_*.tex` | Class means, ablation, method comparison |
| `benchmark_plots_expanded.png` | Length–RMSD + class boxplot |
| `benchmark_ablation_plot.png` | Ablation bar chart |
| `benchmark_long100_time.png` | 10k–50k timing scatter |
